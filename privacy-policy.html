eBay Orders Dashboard Pro Privacy Policy for Chrome Web Store Extension Effective: June 2025 Version: 1.2 Scope: ebay.co.uk · ebay.com · ebay.com.au 🛡️ Overview eBay Orders Dashboard Pro is a Chrome browser extension built for eBay sellers. It enhances the eBay Seller Hub Orders page with a real-time dashboard showing order details, buyer information, item data, and financial summaries. This Privacy Policy explains what data the extension accesses, how it is used, and your rights as a user. We are committed to full transparency and do not collect, sell, or misuse your data. 📦 Data the Extension Accesses While you are on your eBay Orders page, the extension reads the following data from eBay's website to display it inside the dashboard: Order IDs — to fetch and display each order Buyer name & shipping address — shown in order cards, used for copy/export Phone number — displayed per order card (if provided by eBay) Item title, SKU, variation, quantity — shown in order details Order earnings, fees, ad fees — shown as financial summary per order Transaction IDs & Item IDs — used for linking and copy functions eBay username — saved locally for multi-account Supabase sync ⚙️ How Data is Used All accessed data is used solely for the extension's core purpose: displaying and managing eBay orders within your browser session. Order data is displayed in the dashboard for the current browser session only Clipboard copy functions send data only to your local clipboard CSV export saves data only to your local device Bought/unbought status is saved locally in your browser (localStorage) If you connect Supabase, order data is synced to your own private database Your eBay username is stored locally only for session identification 🔑 Permissions Explained clipboardWrite Allows copying order data (address, SKU, earnings) to your clipboard when you click Copy buttons. storage Saves your preferences, Supabase credentials, and bought/unbought status locally in your browser. No external transmission. ebay.co.uk / ebay.com / ebay.com.au Required to read order data from your eBay Seller Hub session and fetch order detail pages. supabase.co (optional) Only used if you manually enter your own Supabase URL and key to sync order status across devices. ☁️ Optional Supabase Cloud Sync The extension offers an optional feature to connect your own Supabase project for syncing order bought/unbought status across devices or accounts. Supabase integration is 100% optional — the extension works fully without it You must manually enter your own Supabase URL and anon key Data is sent only to your private Supabase database — not to any developer server We have no access to your Supabase project or its data You can disconnect or delete your Supabase config at any time from the settings panel 🚫 What We Do NOT Collect We do not transmit any data to the extension developer's servers We do not track browsing history or activity outside eBay order pages We do not use analytics, tracking pixels, or fingerprinting We do not sell, share, or trade any user data with third parties We do not access eBay passwords, payment methods, or account credentials We do not run on any pages other than eBay Seller Hub order pages ✅ Developer Certifications ✓ We do not sell or transfer user data to third parties, outside of approved use cases. ✓ We do not use or transfer user data for purposes that are unrelated to this extension's single purpose. ✓ We do not use or transfer user data to determine creditworthiness or for lending purposes.
